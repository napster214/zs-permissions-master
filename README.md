项目简介：
    本项目基于springboot2.0开发,参照rbac规范设计权限系统
项目说明：
    主要简化中小型企业开发权限系统，达到开箱即用的效果
项目规划：
    暂时规划三个版本发布
    1.0版本
        实现菜单级权限控制
    2.0版本
        实现按钮级权限控制,maven多模块应用
    3.0版本
        实现简单数据级权限，以及基于oauth2协议单点登录
